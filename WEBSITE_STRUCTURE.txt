KEY CONTENT THEMES
==================

Primary Value Proposition:
  Hand-painted watercolour keepsakes for guests + couple portrait, painted live during the event
  on archival A5 cotton paper. Creates a memorable focal point and lasting memento.

Core Service Variants:
  1. Live on the day (base: 3 hours, ~24 portraits, $850+)
  2. Studio commissions from photos (4 weeks turnaround, $400–$500+)
  3. Add-ons (extra hours, family portraits, stationery, bouquet illustrations)

Target Audience:
  Wedding couples (primary), corporate event planners, private celebration hosts

Geographic Scope:
  Melbourne and Sydney metro (travel included)
  Regional Victoria and destination weddings (on enquiry)

Credibility Markers:
  * 10+ years painting experience
  * Professional materials (archival paper, lightfast pigments)
  * Public liability insurance
  * Testimonial from past clients
  * Gallery of portfolio work
  * Transparent pricing model with retainer system

Booking Model:
  50% retainer to secure date + balance due 2 weeks before event
  Payment and terms confirmed in booking agreement


STRUCTURAL FEATURES
===================

Layout Pattern:
  * Full-bleed, immersive single-page scroll
  * Two-column grid on desktop (Why + About Me sections)
  * Responsive grid gallery with mixed tile sizes
  * Sticky header with scroll progress indicator

Visual Hierarchy:
  * Large, hand-painted headline typography
  * Numbered sections (01–05, 01–03) for easy scanning
  * Feature tiles (2×2 on desktop) for emphasis
  * Testimonial card embedded in gallery flow
  * Decorative watercolour-inspired background elements

Interactive Elements:
  * Mobile navigation menu (drawer)
  * Gallery lightbox (tap/click to enlarge)
  * Form submission with success confirmation
  * Scroll-triggered animations and reveals
  * Magnetic button interactions
  * Marquee scrolling text band

Accessibility Considerations:
  * Semantic HTML structure
  * Alt text for all images
  * Keyboard navigation support
  * Reduced-motion safe animations
  * Noscript fallback with essential info and contact email
